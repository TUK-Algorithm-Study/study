# BOJ - 11403

[11403번: 경로 찾기](https://www.acmicpc.net/problem/11403)

### 문제

- 난이도 : Silver 1
- 알고리즘 : 플로이드-워셜
- 입력
    - 첫째 줄 : 정점의 개수 N (1 ≤ N ≤ 100)
    - 둘째 줄부터 N개의 줄 : 그래프의 인접 행렬

### 접근

- **플로이드-워셜** 알고리즘으로 접근
- i → j 로 갈 수 있는 길이가 양수인 경로가 있는지 없는지 구하는 문제이다.
- 플로이드-워셜 알고리즘에서 도달이 가능하면 1로 변경해주면 된다.
