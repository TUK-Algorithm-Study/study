
# BOJ - 1863

[1863번: 스카이라인 쉬운거](https://www.acmicpc.net/problem/1863)

### 문제

- 난이도 : Gold 4
- 알고리즘 : 스택
- 입력
    - 첫째 줄 : n
    - n개의 줄 : 고도가 바뀌는 지점의 좌표 x와 y
- **히스토그램과 유사**

### 접근

- 스택으로 접근
- x좌표를 1부터 n까지 가면서, peek값과 비교하면서 처리한다.
- 만약 peek > y
    - pop( ), count++
- peek == y
    - 종료
- peek < y
    - y값을 집어넣는다.

- 마지막에 0이라는 y 값을 집어넣어서 모든 건물의 높이를 구해준다.
